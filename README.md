# Todo-App
A simple command line todo app with python

* [contributing](https://github.com/legit-programming/Todo-App/blob/main/CONTRIBUTING.md)
#### Functions:
* Add a task to do
* Remove a task to do
* Mark a task to do as done
* Mark a task to do as not done
* Edit a task to do
* List the tasks to do
* List the tasks done
